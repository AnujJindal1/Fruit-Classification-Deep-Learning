# Fruit Classification using Deep Learning

This is a university-level Deep Learning project developed as part of the  
**BCA (Artificial Intelligence & Data Science)** program.

## Project Overview
- Image classification using Convolutional Neural Networks (CNN)
- Transfer learning with ResNet34 architecture
- Cleaned image dataset with 20 fruit classes
- Probability-based predictions
- Live demo using Gradio

## Dataset Details
- Source: Fruits-360 (Kaggle)
- Dataset cleaned by merging similar fruit varieties
- Final dataset contains 20 fruit categories:
  Apple, Banana, Mango, Orange, Grape, Strawberry, Pineapple, Watermelon,
  Papaya, Kiwi, Guava, Pear, Peach, Plum, Cherry, Lemon, Pomegranate,
  Lychee, Avocado, Blueberry
